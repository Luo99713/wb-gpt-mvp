# WB 选品分析工具

用于分析 Wildberries 搜索请求 Excel 表，辅助判断指定小类下的产品机会。

## 当前最新版本

- 最新版本：`v0.5.1`
- 版本目录：`tools/wb-selection-mvp/versions/v0.5.1/`

## 工具定位

这不是全自动选品工具，而是一个“指定小类后深度分析”的工具。

主要回答：

- 这个小类下哪些需求词簇更值得做；
- 哪些属性组合搜索量高、下单量高；
- 哪些词是高搜索低下单，可能存在供给不匹配；
- 哪些词是低搜索高下单，适合小批量测试；
- 品牌词是否显著影响结果；
- 词典覆盖率和待补词情况。

## 核心功能

- 读取 WB 原始搜索请求 Excel；
- 支持普通分析和品牌维度分析；
- API 自动生成类目维度草稿，用户可修改；
- API 生成正式中文属性词典与俄语别名；
- 本地规则处理全表，输出核心词簇、详细词簇、单特征表现、关键词明细；
- 新增 `下单/搜索` 与 `搜索订单对比`；
- 支持多类目批量分析，前提是这些类目已经建立过词库；
- 输出 HTML 报告、多子表 Excel、summary JSON。

## 推荐使用流程

1. 在 WB 后台选择一个具体小类；
2. 导出搜索请求 Excel；
3. 第一次分析该小类时，先生成/调整维度；
4. 确认维度后生成词典；
5. 工具输出报告；
6. 同类目后续复跑时直接使用缓存；
7. 如果需要看品牌影响，切换到品牌维度分析。
