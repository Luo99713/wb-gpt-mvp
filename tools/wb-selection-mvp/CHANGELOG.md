# WB 选品分析工具变更记录

## v0.5.1

- 重构入口逻辑：从 CMD 数字菜单改为 Python 图形窗口。
- 修复 API 设置保存问题，支持保存并回读校验 API Key。
- 增加实时运行日志，避免“选择完没反应”。
- 保留普通分析、品牌维度分析、维度生成/编辑、多类目批量分析。
- 保留 `下单/搜索` 和 `搜索订单对比` 字段。
- 结果统一保存到 `results/`。
- 输出文件名包含类目名和时间。
- 生成 HTML 报告、多子表 Excel 和 summary JSON。

## v0.4.x 主要迭代

- 增加 API 生成类目词典。
- 增加用户修改维度草稿流程。
- 增加品牌维度分析。
- 增加多类目批量分析。
- 增加百分比格式、HTML表格分页、长文本折叠。
- 修复多次 API、HTML 空表、结果路径不清晰等问题。

## v0.3.x 主要迭代

- 从单类目静态词典版本扩展到可生成 HTML 和多子表 Excel。
- 增加结果保存路径、页面内查看所有子表。

## v0.2.x 主要迭代

- 从 Python 依赖运行改为更易启动的版本。
- 逐步修复 Windows 路径、中文乱码、启动脚本问题。

## v0.1.x 主要迭代

- 女包类目 MVP 原型。
- 支持核心词簇、详细词簇、单特征表现、关键词明细、待补词典。
